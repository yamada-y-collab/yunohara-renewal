# 癒の原整骨院 WordPressテーマ 導入手順

静的サイト（プロトタイプ）を、オリジナルWordPressテーマ化したものです。
GitHub Pages（静的）ではPHPが動かないため、**PHPが動くWordPress環境（レンタルサーバー等）** が必要です。

---

## 同梱物

```
wordpress/
├─ yunohara-seikotsuin/          ← WordPressテーマ本体（このフォルダをアップロード）
│   ├─ style.css                 テーマ識別ヘッダー
│   ├─ functions.php             設定・CPT「コラム」登録・アセット読込・構造化データ
│   ├─ header.php / footer.php   共通ヘッダー・フッター・固定CTA
│   ├─ front-page.php            トップページ
│   ├─ page-templates/           固定ページ用テンプレート（下記）
│   │   ├─ menu.php  oxygen.php  es5000.php  accident.php  symptoms.php
│   ├─ archive-column.php        コラム一覧
│   ├─ single-column.php         コラム記事
│   ├─ index.php / 404.php       フォールバック
│   └─ assets/                   css / js / img
├─ yunohara-column-import.xml    コラム3記事のインポートデータ（WXR）
└─ README-WordPress導入手順.md   このファイル
```

---

## 導入ステップ

### 1. テーマを設置
- `yunohara-seikotsuin` フォルダを ZIP 化 → 管理画面「外観 > テーマ > 新規追加 > テーマのアップロード」
  - もしくは FTP で `wp-content/themes/yunohara-seikotsuin/` に直接アップロード
- 「外観 > テーマ」で **「癒の原整骨院」を有効化**

### 2. パーマリンク設定（重要）
- 「設定 > パーマリンク」を開き、**「投稿名」** などを選択して保存
  - ※ これをしないとコラム（/column/…）や固定ページのURLが 404 になります。一度保存するだけでOK。

### 3. 固定ページを作成し、テンプレートを割り当て
「固定ページ > 新規追加」で以下を作成し、**右側「ページ属性 > テンプレート」** を割り当て、**スラッグ**を合わせます。
本文は空でOK（デザインはテンプレート側に入っています）。

| ページタイトル | スラッグ | テンプレート |
| --- | --- | --- |
| 施術メニュー・料金 | `menu` | 施術メニュー・料金 |
| 酸素カプセル | `oxygen` | 酸素カプセル |
| 特殊電気治療 ES-5000 | `es5000` | 特殊電気治療 ES-5000 |
| 交通事故・むち打ち | `accident` | 交通事故・むち打ち |
| 対応症状 | `symptoms` | 対応症状 |

> スラッグはヘッダーナビ・フッターのリンク先と一致させてください（必須）。

### 4. トップページの設定
- 「設定 > 表示設定 > ホームページの表示」＝「固定ページ」にしてもよいですが、
  本テーマは **`front-page.php`** を用意しているため、特別な設定なしでトップに反映されます。
- サイトタイトル（「設定 > 一般」）を **「癒の原整骨院」** にしてください（ロゴ・フッターに反映）。

### 5. コラム記事のインポート
- 「ツール > インポート > WordPress」（Importerプラグインを促されたら導入）
- `yunohara-column-import.xml` を選択してインポート
  → コラム3記事（交通事故テーマ）と「コラム」メニューが入ります。
- 記事内の画像はテーマ同梱の画像を参照しています。差し替える場合は各記事の本文・アイキャッチを編集してください。

### 6. メニュー（任意）
- ヘッダー／フッターのナビは**テーマ側に固定実装**済みのため、メニュー作成は不要です。

## 推奨プラグイン（計画書準拠）
- SEO：**SEO SIMPLE PACK**（各ページのタイトル/メタディスクリプション設定。※静的版のSEO文言はこれで再設定してください）
- フォーム：**Contact Form 7** または **Snow Monkey Forms**（「相談」フォーム設置時）
- セキュリティ：**SiteGuard WP Plugin**
- バックアップ：**UpdraftPlus**

---

## 院側で更新できること
- **コラム追加**：管理画面「コラム > 新規追加」（カテゴリ：交通事故／酸素カプセル／症状別 等）
  - アイキャッチ画像を設定すると一覧・関連のサムネに反映（未設定時はカテゴリ別の既定イラスト）
- **電話・LINE・受付時間など**：`functions.php` 冒頭の定数（`YNH_TEL` / `YNH_LINE_URL` / `YNH_IG_URL`）と、`header.php`／`footer.php` の受付時間テキストを編集
- **写真差し替え**：`assets/img/` を差し替え、または各ページテンプレートを編集

---

## 注意・残作業
- **per-pageのSEO（title/description）と一部FAQ構造化データ** は、静的版では各HTMLの `<head>` に直書きでした。WP版では **SEO SIMPLE PACK で各ページに設定** してください（トップのMedicalClinic/LocalBusiness構造化データはテーマが自動出力します）。
- **旧URL（*.html）→ 新URL の301リダイレクト** を、公開時にサーバー（.htaccess等）かRedirectionプラグインで設定してください。
- 固定ページのスラッグは表のとおりに（ナビのリンク先と一致が必須）。
